Get Started

Installation

Set up SonicVault in minutes

Terminal 
# Clone the repository
git clone https://github.com/ROMAN-AHMAD-NAZAR/-Sonic-vault-Cryptography-Stegnography-.git
cd -Sonic-vault-Cryptography-Stegnography-

# Install dependencies
pip install -r requirements.txt

# Encode a secret message
python cli.py encode "Your secret message" output.wav --password YourPassword --theme rain

# Decode the message
python cli.py decode output.wav --password YourPassword
